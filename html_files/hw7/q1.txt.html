1    - In this question, we want to implement an "aggresive" action - closing the antivirus process 
2    (and by that, making sure it doesn't recognize our malware activity) 
3    - As a first step, we will have to find the antivirus process ID. 
4    - We can use ps with -C flag (command name flag), searching for the specific name of the binary associated with  
5    a process. We will use ps -C "antivirus" to find process associated with the antivirus binary 
6    - Parsing the output of our command - we will use "str.split" to split using the single space (" ") as a delimiter. 
7    - The antivirus process ID can be found in the first object of the returned list 
8    - As a final step we will use "kill" to send SIG_KILL interupt to stop the antivirus process
