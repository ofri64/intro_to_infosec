1    - The intuition for the scoring function is using the known distribution of letters in english text. 
2    - Then, measure the difference between the distribution of the so called decrypted text and 
3    the this known distribution. 
4    - We will assume the english letters frequency is a discrete distribution as detailed in 
5    Wikipedia (https://en.wikipedia.org/wiki/Letter_frequency) 
6    - We will compute the decrypted text distribution simply by counting how many times a letter 
7     appears in the decrypted text and normalize with dividing by the text lengh. 
8    - In order to measure the difference between the two distributions we will use chi-square distribution 
9    (without actually performing the chi-square test, just using the statistic's value. 
10    reference can be found here https://en.wikipedia.org/wiki/Chi-squared_test) 
11   - Identical distribution will get value 0 and the value will grow as the distributions differ. 
12   - So we will setup our score function as -1 * chi-square value, representing the difference between 
13   the theoretical distibution and the empirical one. This way the maximum value in bounded to 0 in case of 
14    identical distributions. 
15   - Another optimization would be to add a penalty for every non-printable char in our candidate decrypted text. 
16   Using a penalty of -0.3 for each such char (probably want to value of penalty to be an hyperparameter that 
17    can be sadjusted)
